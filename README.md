## 部署到 GitHub Pages

1. Fork 或建立新 repo
2. 把 `index.html` 上傳到 repo 根目錄
3. 到 **Settings → Pages → Source** 選 `main` branch，資料夾選 `/ (root)`
4. 等待約 1 分鐘，即可透過 `https://你的帳號.github.io/repo名稱/` 存取
