1.1風險的定義與要件

風險三要件：①結果具不確定性（非確定性）、②有損失發生的可能、③屬於將來性

陷阱「結果必須為確定性」→ 錯。「損失金額必須龐大且可金錢衡量」→ 錯（不限龐大，也含無形損失）。

損失：有形（可金錢衡量）+ 無形（精神痛苦）。保險可保之損失＝因意外事故造成。

1.2風險分類體系（六大分法）超高頻
分類法類型 A類型 B
損失性質純粹風險：只有損失，無獲利投機風險：可能損失也可能獲利
影響範圍基本風險：波及大、難控制（地震）
純粹+投機兼有		特定風險：只影響個人、易控制
常屬純粹風險
可衡量性客觀風險：可統計、不受人為故意影響主觀風險：因人而異
變動性靜態風險：與社會變動無關動態風險：受社會經濟政治影響

必背 天然災害（地震）＝ 客觀＋純粹＋靜態＋基本風險

基本風險是唯一兼有純粹與投機雙重性質的分類。

陷阱「特定風險與經濟、政治、天然災害有關」→ 錯，那是基本風險。特定風險只影響個人。

1.3損失頻率 vs 損失幅度經典陷阱
損失頻率＝一定期間內每一風險單位發生事故的平均「次數」
損失幅度＝每一次損失事故的平均「金額」（又稱損失額度）
損失成本＝ 頻率 × 幅度

陷阱 題目最愛將兩者定義互換！「平均損失金額」→ 幅度，「平均次數」→ 頻率。

幅度高幅度低
頻率高特別嚴重風險重要風險
頻率低重要風險不重要風險
1.4風險管理四步驟與五方法

四步驟：風險確認 → 風險衡量 → 風險方法選擇 → 決策執行與評估

陷阱「風險衡量」屬第二步，非第一步。

方法時間點說明經典例題
避免事前最簡單經濟安全不去夜市/觀光景點
預防事前降低發生機會高血壓少油少鹽、勤洗手戴口罩
抑制事中/事後減輕損失程度發燒去篩檢、自主管理
移轉事前轉嫁給保險公司購買保險（非年金型）
自留—自己承擔保額不足的部分

陷阱 購買年金型保險 ≠ 移轉，屬「儲蓄投資以備未來所需」。

陷阱「高血壓不吃太油膩」→ 事前預防，非事中抑制。

風險自留原因（四個全都是）：①疏忽不處理 ②自己承擔較划算 ③無適當方式 ④有能力承擔

1.5風險因素三分法
類型性質定義舉例
實質風險有形物理條件增加風險老舊建築
道德風險無形故意不當得利縱火騙保
心理風險無形疏忽/不小心投保後不維護房子

記法：道德＝故意＝犯罪級；心理＝疏忽＝懶人級

1.6保險稅務優惠

①保費一定金額內可自綜合所得額扣除 ②有指定受益人 → 死亡保險金不計入遺產 ③未指定受益人 → 計入遺產 ④滿期保險金免課所得稅

陷阱「滿期保險金不用課遺產稅」→ 表述不當，滿期金跟遺產稅無關，正確是免所得稅。

人身風險事故導致：遺族生活費用增加、老年退休費用增加（非減少）、醫療費用增加、收入減少。

4.1保險費結構與三差損益
保險費 ＝ 純保險費 ＋ 附加費用
純保險費：由預定死亡率＋預定利率計算（保障成本）
附加費用：營業費用、佣金（預定營業費用率）

三差損益：利差益（投資報酬 > 預定利率）、死差益（實際死亡率 < 預定）、費差益（實際費用 < 預定）

保單紅利利源＝利差益＋死差益＋費差益之和，分紅回饋保戶。

4.2預定利率與保費關係

反向關係：預定利率↑ → 保費↓（預期投資收益高，少收保費即可）

自民國89年11月1日起考量：①險種特性 ②過去資金運用績效 ③未來投資規劃 ④社會經濟發展趨勢，壽險公司可自行訂定預定利率。

4.3生命表
類型編製者基礎
國民生命表內政部全國國民人口普查
經驗生命表壽險公會被保險人實際經驗

英國衡平保險社首先根據生命表計算合理保費 → 現代人壽保險制度建立。

4.4責任準備金與解約金

責任準備金＝純保費扣除已過危險保費後提存保管，按險種計算記載於特設帳簿。

保險法各種準備金：賠款準備金、未滿期保費準備金、特別準備金（陷阱無「差額準備金」）

解約金：躉繳/分期繳費期間解約金 ≤ 保單價值準備金；繳費期滿後解約金 ＝ 保單價值準備金。

保單紅利「儲存生息」方式 → 被保險人死亡時保險金額與原保險金額一樣（不增不減）。

利率變動型年金累積期間：保費 − 附加費用 → 依宣告利率計算保單價值準備金。

4.5人身保險構成三原則

①互相扶助的觀念 ②公平的危險分擔 ③收支相等的原則

陷阱「損失填補的作用」→ 非人身保險構成原則（屬財產保險概念）。

保險是集合大多數人力量分散風險消化損失的社會互助經濟保障制度，是交換價值相等的契約（對全體投保者而言）。

5.1年金保險分類出現19次
類型說明
生存年金被保險人生存期間持續給付
確定年金不以生存為條件，約定期間一定給付

5.3投資型保險

核心特色：盈虧自負、彈性繳費、分離帳戶

全權委託投資標的：公債/國庫券、臺灣存託憑證、各國中央政府公債

陷阱結構型商品＝非全委。這是超常考陷阱。

費用三類：①保險費用 ②保單管理費 ③投資相關費用（申購手續費、贖回費等）

專業課程測驗辦理：保發中心、主管機關認可之保險相關單位、壽險公會（皆可）

投資型保險帳戶價值有高低起伏但可能降低至零。

5.4長期照顧保險

免責期 ≤ 6 個月　巴氏量表 ≥ 3 項障礙　認知功能：MMSE評估，CDR達中度以上

六項日常能力：進食、移位、如廁、沐浴、平地移動、更衣

陷阱 109年11月金管會取消認知功能障礙相關舊規定。CDR需「中度以上」非輕度。

5.5傷害保險

失能保險金 ≤ 保險金額（不論次數及程度）

事故發生日起 180 天 內致死亡/失能 → 保險公司負保險責任

日額型每次給付 ≤ 180 日（超過但可證明因果關係者不在此限）

傷害保險之保費：適用寬限期間、保險人得訴訟請求、利害關係人得代繳

陷阱 年金保險的三項特殊規定不適用於傷害保險！

生存保險：被保險人在保險期間內死亡 → 保險公司不負給付責任，但退還已繳保費。

5.6微型保險與外溢保單

微型保險承保對象：①農民保險之農民 ②低/中低收入戶 ③身心障礙者 ④低收入老人 ⑤原住民 ⑥漁民

認定文件：中低收入證明、身心障礙手冊、漁船船員手冊、原住民戶籍

陷阱離島居民戶籍是居住所證明，非資格認定文件。

外溢保單：穿戴裝置 → 改變生活作息 → 提升健康 → 啟動減費機制或健康促進獎勵金

實支實付醫療保險張數上限 3 張，不適用於微型保險、旅平險、學生團體保險。

7.1保單活化制度高頻

定義：用原保單保單價值準備金轉換為醫療/長照/年金保險。屬功能性轉換（他益型→自益型）。

轉換後預定利率變低 → 保費成本提高。部分保單解約扣稅後可能比轉換更划算。

需要保人及被保險人親自簽章（非僅要保人）。

陷阱「保單活化對客戶百利無一害」→ 錯。可能領到的總額低於轉換前。

不得回復原契約：①已開始給付/有理賠紀錄 ②原保險事故已發生 ③轉換生效日起 3年 後才主張

變更適合度評估：確認對象含要保人及被保險人。

7.2保障需求分析與行銷

保障需求：①遺族生活資金（以丈夫死亡時妻子平均餘命計算）②晚年生活 ③教育基金 ④購屋 ⑤借款償還金

應急預備金：醫療/喪亡處理/借款償還/臨時預備金。陷阱「子女教育金」→ 非應急預備金。

保費原則：年收入 1/10；保額參考：年收入 10倍

行銷通路市占：壽險公司 49% > 銀行保險 37% > 經紀人

行銷過程：接近→問題→解決→決心→滿意（口訣：接問解決滿）

招攬程序：蒐集資料/發現問題 → 提示對策 → 成交/售後

售後服務開端＝遞交客戶保單

保險存摺：保險商業同業公會推出，查詢保單。可看保障缺口、申請理賠、查詢各保險公司保單。

保險經紀人＝向承保保險業收取佣金之人。

業務員同時擁有人身/財產保險資格 → 仍不得同時為壽險公司及產險公司招攬。

招攬文件保存 ≥ 契約期滿或不同意承保後 5 年

團體保險保費可自薪津扣除、可提高繼續率。但仍須經核保評估。

保險業除傳統通路外的行銷管道：郵寄行銷、電話行銷、職域/關係企業行銷、架設網路平台行銷

某甲投保每年複利增值3%之20年養老保險，保額20萬，第二年死亡：20萬 × 1.03 × 1.03 ＝ 212,180
