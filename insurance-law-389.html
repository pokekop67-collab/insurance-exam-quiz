第一章 · 保險契約基本概念
猜(3) 命中率 39%
1.1四大關係人高頻
角色定義重點
保險人承擔風險的保險公司經營保險事業之各種組織
要保人有保險利益，申請契約並負繳費義務契約當事人、須有行為能力
被保險人以生命或身體為保標的之人要保人須對其有保險利益
受益人事故發生時有權受領保險金由要保人指定/變更、無人數限制

保險利益存在於要保人與被保險人之間（非受益人）。

要保人對以下人有保險利益：①本人或家屬 ②生活費/教育費所仰給之人 ③債務人 ④管理財產/利益之人

陷阱「債權人」（借錢給你的人）→ 無保險利益。

限制行為能力人訂立契約 → 須經法定代理人同意或於限制原因消滅後自己承認。

要保人對被保險人所知者應通知保險人。

1.2受益人規定

失能/醫療保險受益人 → 以被保險人本人為限，不得更改。

受益人指定/變更 → 要保人之權利，無人數及對象限制。要保人得於事故發生前以契約或遺囑處分。

受益人故意致被保險人死亡 → 喪失受益權。

受益人故意致被保險人未致死僅致失能 → 保險公司仍給付全額失能保險金。

利害關係人均得向保險人請求保險契約謄本。

人身保險契約不得為指示式或無記名式（僅財產保險可以）。

1.3損害填補、分攤原則、複保險、共保

損害填補：防止被保險人不當得利。適用於財產保險＋實支實付健康險。一張保單即可發生。

分攤原則：使被保險人不致不當得利（非要保人）。由損害填補延伸。

複保險：同一保險利益、同一事故與數保險公司分別訂立數個契約。故意不通知 → 契約無效。

共保方式：①巨災損失 ②配合政策 ③公共利益 ④提升服務 ⑤主管機關核准

1.4未滿15歲與受監護宣告者

未滿15歲投保含死亡給付人壽保險：除喪葬費用外，其餘死亡給付於滿15歲時始生效力。

喪葬費用保險金額 ≤ 遺產稅喪葬費扣除額之一半。

受監護宣告者投保人壽保險：超過部分不負給付責任，無息退還超過部分已繳保費。

國外投資上限 45%（§146-4）。不動產投資上限 30%（除自用，§146-2）。

第二章 · 保險法核心條文與契約條款
🏆 全卷之王 · 猜(3) 29% · 選(4)最冷 16%
2.1保費繳納與寬限復效超高頻

繳費方式：一次繳清（躉繳）/ 分期（年/半年/季/月繳）。保費確定者 → 契約生效前交付。

繳費期間與保險期間不一定相同，可較短。

利害關係人均得代要保人交付保險費（§115）。

停復效時間軸：
保費到期 → 寬限期間（30日）→ 未繳 → 停效
停效後 ≤6個月 → 直接申請復效
停效後 >6個月 → 保險人 5日 內要求可保證明 → 收到後 15日 不拒絕 → 視為同意

必背：5日要求、15日不拒＝同意。

2.2契約效力：撤銷、解除、終止高頻
效力時機/事由效果
撤銷收到保單翌日起 10日 書面翌日零時起自始無效（郵寄以郵戳當日零時）
解除違反告知義務等法定事由自始不存在（溯及），保險人無須退還保費
終止保險人破產、保標的物完全滅失自終止日起消滅；終止前事故仍賠

陷阱「解除為自解除日起消滅」→ 錯，解除是自始不存在。

陷阱「無體檢不得主張撤銷權」→ 錯。無論有無體檢均享有。

法定終止事由：保險人破產、保標的物非因契約所載事故完全滅失（不含要保人任意終止、停效逾期）。

破產中止 → 終止後保費已交付者，保險人應返還。

2.3契約無效事由

法定無效事由（全部都是）：

① 契約訂立時保標的危險已發生或已消滅
② 由第三人訂立死亡保險，未經被保險人書面同意並約定保險金額
③ 真實年齡超過保險人所定保險年齡限度
④ 要保人意圖不當得利而為複保險

約定無效（顯失公平）：①免除/減輕保險人義務 ②要保人/受益人/被保險人拋棄或限制權利 ③有重大不利益

陷阱「加重保險人之義務」→ 非約定無效事由。

2.4保證續保、電子保單、保單條款

保證續保：按續保生效時陳報主管機關之費率及被保險人年齡重新計算（不看身體狀況、不用經驗費率）

電子保單：①保險人與要保人共同約定 ②約定後仍可恢復紙本 ③保費應反映成本

陷阱「約定後不能恢復紙本」→ 錯。

保單條款以網址取代紙本：是保險人與要保人共同遵守的約定（非被保險人）。節省成本需於附加費用反映。

示範條款目的：統一作業方式、減少分歧、確立信譽。陷阱「授權訂定彈性提高競爭力」→ 非目的。

2.5受益人代繳、告知義務、危險增加

受益人代繳保費：屬民法無因管理、不影響其他受益人保險金權利、不得要求排除其他受益人

告知義務違反 → 保險人得解除契約，無須退還保費。

非因要保人/被保險人行為致危險增加 → 未及時通知 → 對保險人損失應負賠償。

保險人對因履行道德上義務所致損害 → 負賠償責任。

僅要保人知悉危險已發生 → 保險人得請求償還費用，已收保費無須返還。

特約條款：當事人違背 → 他方得解除契約（即使危險已發生）。但須故意/過失才可據為解約。

2.6各類保險專章差異
寬限期間訴訟請求保費利害關係人代繳
人壽保險適用得得
傷害保險適用得得
健康保險適用得得
年金保險不適用不得得

年金三特殊：不適用寬限期、不得訴訟請求保費、利害關係人得代繳。

由第三人訂立之年金保險，未經被保險人書面同意並約定保險金額 → 契約仍有效（與死亡保險不同！）

傷害保險事故日起 180天 內致死亡/失能 → 保險公司負責。

3.1遺產稅扣除額（必背數字）超高頻
項目扣除額
配偶400 萬
父母（每人）100 萬
祖父母（每人）40 萬
成年子女（每人）40 萬
未成年子女40萬 + 每差1歲加40萬

例：遺有成年子女1名及母親 → 40+100＝140 萬

3.2保險金與遺產稅的關鍵規則
有指定受益人 → 死亡保險金 不計入遺產
未指定受益人 → 死亡保險金 計入遺產

被保險人早於受益人身故 → 保險金歸受益人。受益人再身故 → 成為受益人之遺產（非被保險人遺產）。

贈與稅：夫妻間贈與不計入贈與總額。婚嫁贈與 ≤ 100 萬 也不計入。

例：老王贈子房屋2000萬＋贈妻轎車200萬＋贈女出嫁禮金80萬 → 課稅贈與總額＝2000 萬（夫妻免計、婚嫁80萬<100萬免計）

遺產稅繳納方式：①現金 ②實物抵繳 ③分期繳納

3.3保費扣除額

每人每年保費列舉扣除額 24,000 元（全民健保費不受此限）

可扣除的保費：為本人、配偶、受其扶養之直系血親尊親屬繳納之壽險保費

陷阱 以下不可扣除：①兄弟姊妹的壽險保費 ②非同一申報戶 ③境外保單（未經核准在台銷售）④非人身保險（汽車/火災/責任保險）

3.4繼承順序與特留分

法定繼承順位：①直系血親卑親屬（子女）②父母 ③兄弟姊妹 ④祖父母。配偶＝當然繼承人。

母與父離婚 → 子女仍對父母有繼承權。男子入贅 → 保有對父母繼承權。繼父母與繼子女 → 不得相互繼承。

第四章 · 金融消費者保護與合規
猜(1) 27%
4.1金融消費者保護法

契約條款顯失公平 → 該部分無效。條款疑義 → 有利於金融消費者解釋。

金融服務業責任不得預先約定限制或免除，違反者無效。

金融教育宣傳不得引薦個別金融商品。

評議成立且經法院核可後：不得再起訴、不得再評議。如有無效原因得提起撤銷之訴。

酬金制度：不應僅以業績為基礎，應納入稽核缺失、客訴、KYC等非財務指標。

陷阱「一賣出就發高額酬金」→ 不符衡平原則。

4.2洗錢防制（低風險交易情境）

低洗錢風險（可簡化審查）：①不具保單價值準備金之保險 ②微型/小額終老保險 ③團體年金/定壽/傷害/健康/醫療保險 ④客戶無高風險且年繳 ≤3萬（躉繳 ≤7.5萬）⑤強制車險/農業保險/小額財產保險（≤5萬）

陷阱「具有保單價值準備金」→ 非低風險。「結構型商品」→ 非低風險。

4.3定型化契約與審閱期、公平待客、個資法

定型化契約：企業單方面預先擬具（非雙方共同議定）。消費者僅有簽與否的權利。

壽險商品審閱期 ≥ 3 日。陷阱「保戶可主動放棄審閱權」→ 不可。未提供審閱期 → 定型化契約無效。但有提供者仍須提供契約撤銷權。

公平待客：9大原則＋第10項（董事會重視）。訓練每年 ≥ 3小時（陷阱非2小時）。未完成 → 撤銷登錄。

個資法目的：規範蒐集、處理及利用，兼顧權益保護與合理利用。陷阱「禁止蒐集或無限制保護」→ 錯。

5.1登錄制度出現19次

每 5 年 完成教育訓練並換發登錄證。

可登錄機構：壽險公司、保險代理人公司、保險經紀人公司、兼營保險經紀/代理之銀行

陷阱「壽險公會」→ 非登錄機構。

不得登錄：①文件虛偽 ②受破產宣告未復權 ③受同類保險停招期限內

陷阱「產險受停招處分」→ 產險非同類，不影響壽險登錄。

再登錄：①註銷後轉任 ②停招期滿後重新登錄

曾受第13條撤銷登錄 → 須重新通過資格測驗。曾受第19條撤銷登錄 → 不可立即重新報名。

法令遵循課程：保發中心製作數位課程。冒名 → 撤銷登錄。不參加商品說明會 → 非撤銷登錄事由。

5.2三種保險中介人
角色立場收入來源
保險經紀人基於被保險人利益向保險公司收取佣金
保險代理人代理保險公司向保險公司收取佣金
保險公證人中立第三方查勘鑑定估價而予證明

產險經紀人業務員取得人身保險資格 → 得無條件登錄另一壽險經紀人公司（同類型才可無條件）。

保險業務員同時有人身/財產資格 → 仍不得同時為壽險及產險公司招攬。

A.2團體保險

團體＝ 5人以上 ＋ 非以購買保險為目的之組織

異動加保：書面通知，自通知到達翌日零時起生效（如起保日在後則以起保日零時）。

危險顯著增加：要保人知悉後 2 週 內通知。保險人 30日 內得要求增加保費或終止。

人數不足：保險人得終止並按日數比例退還未滿期保費。

涉訟 → 不得排除小額訴訟管轄法院。

請領身故保險金：死亡證明＋除戶戶籍謄本、保險金申請書、受益人身分證明。陷阱「印鑑證明」→ 非必要。

請領失能保險金：失能診斷書、保險金申請書、受益人身分證明。陷阱「印鑑證明」→ 非必要。

保險合作社之保險契約以參加保單紅利者為限。保險公司得簽訂參加保單紅利之契約（非以此為限）。

接管期間主管機關得限制：新業務承接、有效保單變更或終止、保單借款、解約金。

涉及外匯業務 → 須經中央銀行許可（非金管會）。

保險金信託：要保人於保險事故發生前預先洽定（非訂立契約時同時洽訂）。
